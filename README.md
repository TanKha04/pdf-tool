# Công cụ Tách & Gộp PDF

Công cụ web miễn phí để tách và gộp file PDF, sắp xếp lại trang, xem trước trước khi tải về.

## Tính năng

- ✅ Gộp nhiều file PDF thành một
- ✅ Tách file PDF thành từng trang riêng
- ✅ Xem trước các trang PDF
- ✅ Sắp xếp lại thứ tự trang bằng kéo thả
- ✅ Chọn trang muốn giữ lại
- ✅ Đặt tên file khi tải về
- ✅ Giao diện đẹp, dễ sử dụng
- ✅ Hoạt động hoàn toàn trên trình duyệt (không upload lên server)

## Cách sử dụng

1. Mở trang web
2. Kéo thả hoặc chọn file PDF
3. Sắp xếp lại thứ tự file nếu cần
4. Chọn chức năng:
   - **Gộp File**: Gộp tất cả file thành một
   - **Tách File**: Tách từng trang thành file riêng
   - **Xem Trước**: Xem, chọn và sắp xếp lại trang trước khi tải về

## Bảo mật

Tất cả xử lý được thực hiện trên trình duyệt của bạn. File không được upload lên server nào.
